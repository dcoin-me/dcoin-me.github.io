Installation on IOS

Dcoin not yet had time to get in the App Store

Therefore, to run Dcoin on devices with IOS, perform the following steps:

  1. Install Xcode 7.x
  2. Install git
  3. Install golang 1.5.1
  4. Connect your iPhone or iPad to your computer
  5. Run Xcode
  6. Top on the menu go to Xcode->Preferences->Accounts. Bottom left click +, then click Add Apple ID and enter your data. Right click View details and then - create near IOS Development
  7. Create a new empty project. Tab General in Deployment Info select Deployment Target corresponding to your version of IOS, further specify the unique Bundle Identifier, for example, dcoin973459 (below will need to use it), in Team select your account and click Fix issue
  8. Perform console #mkdir ~/go-projects; cd ~/go-projects && export GOPATH=~/go-projects && go get -u github.com/c-darwin/mobile/cmd/gomobile && cd $GOPATH/src/github.com/c-darwin/mobile/cmd/gomobile && go build -o $GOPATH/bin/gomobile && $GOPATH/bin/gomobile init && go get -u github.com/jteeuwen/go-bindata/... && go get -u github.com/c-darwin/stoppableListener && go get -u github.com/c-darwin/dcoin-go && cd $GOPATH/src/github.com/c-darwin/dcoin-go/ && $GOPATH/bin/go-bindata -o="packages/static/static.go" -pkg="static" static/... && cd ~ && $GOPATH/bin/gomobile build -target=ios -x -v -name=dcoin973459 github.com/c-darwin/dcoin-go
  9. In your home directory will file Dcoin.app. In xcode go to the menu Windows -> Devices, select your device on the left and on the right click + and select the file Dcoin.app
  10. On your device will Dcoin
