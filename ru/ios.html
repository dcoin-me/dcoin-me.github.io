Установка на IOS

Dcoin еще не успел попасть в Appstore

Поэтому, чтобы запустить Dcoin на устройствах с IOS нужно выполнить следующие шаги:

  1. Установите Xcode 7.x
  2. Установите git
  3. Установите golang 1.5.1
  4. Подключите iPhone или iPad к компьютеру
  5. Запустите Xcode
  6. Вверху в меню зайдите в Xcode->Preferences->Accounts. Слева внизу нажмите +, затем нажмите Add Apple ID и укажите свои данные. Справа нажмите View details и затем - create возле IOS Development
  7. Создайте новый пустой проект. Во вкладке General в Deployment Info укажите Deployment Target соответствующий версии Вашей IOS, далее укажите уникальный Bundle Identifier, например, dcoin973459 (ниже нужно будет его использовать), в Team выберите Ваш аккаунт и жмите Fix issue
  8. Выполните в консоле #mkdir ~/go-projects; cd ~/go-projects && export GOPATH=~/go-projects && go get -u github.com/c-darwin/mobile/cmd/gomobile && cd src/github.com/c-darwin/mobile/cmd/gomobile && go build -o ~/go-projects/bin/gomobile && ~/go-projects/bin/gomobile init && go get -u github.com/c-darwin/dcoin-go && cd ~ && ~/go-projects/bin/gomobile build -target=ios -x -v -name=dcoin973459 github.com/c-darwin/dcoin-go
  9. В Вашей домашней директории появится файл Dcoin.app. В xcode зайдите в меню Windows -> Devices, выберите свой девайс слева, а справа нажмите + и выберите файл Dcoin.app
  10. На Вашем девайсе появится Dcoin
